## v1.2.31 — 修复桌面端升级后看不到本地数据的问题

### 中文

#### 桌面端

- 修复 1.2.30 中桌面客户端升级后可能进入新手引导、看不到原有作品和快照的问题。
- 原有本地数据没有被删除；本次更新会恢复读取原来的本地数据位置。
- 保留 1.2.30 中的启动、AI 设定应用、同步和帮助文档渲染修复。

#### Android 端

- Android 安装包版本号更新为 `1231`。
- 包含 1.2.30 的移动端 AI 会话保持、设定写入和菜单显示修复。

### English

#### Desktop

- Fixed an issue in 1.2.30 where the desktop client could show onboarding and hide existing local works and snapshots after upgrading.
- Existing local data was not deleted; this update restores access to the original local data location.
- Includes the 1.2.30 fixes for startup, AI settings application, sync reliability, and Help document rendering.

#### Android

- Android version code is now `1231`.
- Includes the 1.2.30 mobile fixes for AI chat persistence, settings writes, and conversation menu display.
